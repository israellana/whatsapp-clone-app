### Getting Started

<a name="requisites"></a>
### Prerequisites

Add your Firebase config in Firebase.js constructor, with Firebase bucket path as well.

```
...
  constructor() {
    this._config = {
        //Your Firebase config here :)
    };

    this.init();
  }
...
```

Run the follow commands and the project will open on port 8080.

`$ npm install && npm run start`
